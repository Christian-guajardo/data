Traitement De Data

Définition de Data

En informatique, une donnée(Data) est la représentation d'une information dans un programme : soit dans le texte du programme (code source), soit en mémoire durant l'exécution.
Les données, souvent codées, décrivent les éléments du logiciel tels qu'une entité (chose), une interaction, une transaction, un évènement, un sous-système, etc.
Les données peuvent être conservées et classées sous différentes formes : textuelles (chaîne), numériques, images, sons, etc.
Les données variables qui font la souplesse d'un programme sont généralement lues depuis un appareil d'entrée utilisateur (clavier, souris…), un fichier, ou en réseau.

Le traitement de donner sers a utitiliser ou modifier des donner.
Quand on récupère des donnés nous ne pouvons pas les voir directement mais grace à certain programme nous pouvons les modifier et les mettre sous certain forma.
